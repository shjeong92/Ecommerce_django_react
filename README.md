### 1. cd into frontend forder and type "npm install" and enter to install all of the packages.

### 2. pip install virtualenv 

### 3. type "virtualenv myenv" to create python3 virtual environment

### 4. type "source ./myenv/bin/activate" and enter to activate the virtualenv.

### 5. type "pip install django-cors-headers djangorestframework djangorestframework-simplejwt asgiref Pillow PyJWT  pytz sqlparse" to install all of python packages


### 6. cd into backend folder and type "python manage.py createsuperuser" to make your own admin account
and type "python manage.py runserver" to run backend server

### 7. open another terminal and cd into frontend folder and type "npm start" to run frontend server and
